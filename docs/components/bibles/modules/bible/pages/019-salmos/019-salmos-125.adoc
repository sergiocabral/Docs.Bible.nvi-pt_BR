[[v19-125-0]]
=== Salmos, 125
// nvi ?, NVI - Nova Versão Internacional

[[v19-125-1]]

^<<v19-125-1,1>>^ Os que confiam no Senhor são como o monte Sião, que não se pode abalar, mas permanece para sempre.

[[v19-125-2]]

^<<v19-125-2,2>>^ Como os montes cercam Jerusalém, assim o Senhor protege o seu povo, desde agora e para sempre.

[[v19-125-3]]

^<<v19-125-3,3>>^ O cetro dos ímpios não prevalecerá sobre a terra dada aos justos, se assim fosse, até os justos praticariam a injustiça.

[[v19-125-4]]

^<<v19-125-4,4>>^ Senhor, trata com bondade aos que fazem o bem, aos que têm coração íntegro.

[[v19-125-5]]

^<<v19-125-5,5>>^ Mas aos que se desviam por caminhos tortuosos, o Senhor dará o castigo dos malfeitores. Haja paz em Israel!
