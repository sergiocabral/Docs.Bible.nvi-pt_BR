[[v19-126-0]]
=== Salmos, 126
// nvi ?, NVI - Nova Versão Internacional

[[v19-126-1]]

^<<v19-126-1,1>>^ Quando o Senhor trouxe os cativos de volta a Sião, foi como um sonho.

[[v19-126-2]]

^<<v19-126-2,2>>^ Então a nossa boca encheu-se de riso, e a nossa língua de cantos de alegria. Até nas outras nações se dizia: "O Senhor fez coisas grandiosas por este povo".

[[v19-126-3]]

^<<v19-126-3,3>>^ Sim, coisas grandiosas fez o Senhor por nós, por isso estamos alegres.

[[v19-126-4]]

^<<v19-126-4,4>>^ Senhor, restaura-nos, assim como enches o leito dos ribeiros no deserto.

[[v19-126-5]]

^<<v19-126-5,5>>^ Aqueles que semeiam com lágrimas, com cantos de alegria colherão.

[[v19-126-6]]

^<<v19-126-6,6>>^ Aquele que sai chorando enquanto lança a semente, voltará com cantos de alegria, trazendo os seus feixes.
